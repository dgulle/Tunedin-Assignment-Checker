Tunedin Assignment Checker

Enter your Entra ID app registration details to connect directly to Microsoft Graph.

Your Client ID and Tenant ID are saved in your browser's local storage so you don't have to re-enter them.

How do I create an app registration?
  1. Go to Entra Portal (entra.microsoft.com) > App registrations
  2. Click New registration
  3. Name it (e.g. "Tunedin Assignment Checker")
  4. Set Redirect URI to Single-page application (SPA) with value:
    https://dgulle.github.io/Intune-Assignment-Checker/
  5. Under API permissions, add Microsoft Graph delegated permissions:
    • DeviceManagementConfiguration.Read.All
    • DeviceManagementApps.Read.All
    • DeviceManagementManagedDevices.Read.All
    • DeviceManagementScripts.Read.All
    • Group.Read.All
    • User.Read.All
  6. Copy the Application (client) ID and your Tenant ID into the fields above
